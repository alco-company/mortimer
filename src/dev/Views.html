Overview

The Views::Components::List::Header class is a component that renders a table header.

One typical use case is to render a table header with a label and a sort link. That looks like this: <%= render_header sort: true, label: t('.name'), column: :name %>

Most lists have a checkbox column. That looks like this: <%= render_header column: 'checkbox', css: 'hidden' %>

The edit column is a special case. It looks like this: <%= render_header column: 'edit' %> and offers the user a link to edit the record.
